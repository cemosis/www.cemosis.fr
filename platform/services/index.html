Services

Cemosis propose différents services dans le cadre de collaborations avec les entreprises et les partenaires académiques. Il permet l'organisation des projets de manières cohérente et efficace au travers de différents outils logiciels:

  • Feel++: une librarie de modélisation et simulation développée localement et en collaboration avec des partenaires externes;
  • Git / Github : un environnement de gestion de code source efficace alliant le gestionnaire de version git et le site d'hébergement de code github;
  • Trello : Un outil de gestion de projet se basant sur une organisation sous forme de cartes associées à des tâches.

    • L'ensemble logiciel environnant un projet est détaillé plus amplement dans ce document.

    Cemosis dispose aussi de l'accès à des ressources de calcul propres, telle que le serveur de calcul Atlas, hébergé à l'IRMA. Il permet aussi d'accéder à des ressources de calculs plus importantes situées au mésocentre de l'Université de Strasbourg, notamment au travers du programme HPC-PME.
