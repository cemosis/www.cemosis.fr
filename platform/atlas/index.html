French Version

Irma-atlas est une nouvelle machine de calcul installée à l'IRMA depuis le 1er octobre 2013. La configuration du serveur est la suivante :

  • 64 coeurs sur 4 sockets (AMD Opteron 6386 SE 2,8 Ghz),
  • 512 Go de RAM,
  • 2,4 To de stockage sur SSD,
  • 70 To pour le stockage de données (10 000 tr/min),
  • Montage NFS des données du laboratoire présentes sur d'autres baies,
  • 2 ports PCI-Express 16x et 3 ports PCI-Express en 8x (Possibilité d'y installer des accélerateurs type GPU ou Xeon Phi).

    • Pour la partie logicielle sur cette machine, vous pouvez vous référer aux documentations suivantes:

  • Chroot
  • Licensed Software

    • Vous pouvez procéder à une demande de compte sur la machine. N'oubliez pas d'indiquer les raisons de votre demande.

    English Version

    Irma-atlas, a new computer dedicated to scientific computation, was installed at the IRMA the 1st october of 2013. The server specifications are the following:

  • 64 cores on 4 sockets (AMD Opteron 6386 SE 2.8 GHz)
  • 512 Gb of RAM,
  • 2.4 TB of storage on SSD
  • 70 TB for data storage (10,000 rpm HDD)
  • NFS mount for access to laboratory data,
  • 2 PCI-Express x16 slots and 3 PCI-Express 8x ports (New accelerator devices can be installed, eg GPU or Xeon Phi).

    • For a (french) documentation about the software installed on that server, please have a look to the following pages:

  • Chroot
  • Licensed Software

    • You can ask for an account on that server. Do not forget to motivate your application.
