Machine virtuelle

Afin de tester Feel++, Cemosis vous propose une marche à suivre pour installer (ou télécharger) une machine virtuelle dont l'utilisation est décrite en fin de page.

Marche à suivre pour construire sa propre machine

Nous recommandons l'installation de Feel++ sur debian unstable (Nous recommandons d'utiliser le fichier netinst.iso). Vous pouvez installer Feel++ soit dans une machine physique, soit virtuelle (Virtual Box).

Une fois la machine installée, nous vous conseillons le script suivant (en root) afin de satisfaire toutes les dépendances de Feel++ :

{% highlight bash %} apt-get update apt-get upgrade apt-get install feel++-doc libfeel++-dbg libfeel++-dev libfeel++1 git gmsh libgmsh-dev {% endhighlight %}
